# Fragile States Index (FSI)

**Theme:** Governance & Fragility
**Source:** Fund for Peace
**Geographic Coverage:** Global (179 countries)
**Time Coverage:** 2005 – present (annual)
**Unit of Observation:** Country-year
**Format:** Downloadable CSV, interactive
**Access:** [Fragile States Index Global Data](https://fragilestatesindex.org/global-data/)
**License:** Free to access
**Last Verified:** April 2026

---

## Description

Ranks countries by fragility using 12 indicators across cohesion, economic, political, and
social/cross-cutting dimensions. Widely used in conflict and development research as a
covariate or dependent variable.

## Key Variables

| Variable | Description |
|----------|-------------|
| Total FSI score | Overall fragility (higher = more fragile) |
| C1–C3 | Cohesion indicators (security apparatus, factionalized elites, group grievances) |
| E1–E3 | Economic indicators (economic decline, uneven development, poverty) |
| P1–P3 | Political indicators (state legitimacy, public services, human rights) |
| S1–S2 | Social indicators (demographic pressure, refugees/IDPs) |
| X1 | External intervention |
