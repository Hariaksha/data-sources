# ACLED Conflict Index

**Theme:** Conflict & Safety
**Source:** ACLED (Armed Conflict Location & Event Data Project)
**Geographic Coverage:** Global
**Time Coverage:** Recent (updated periodically)
**Unit of Observation:** Country
**Format:** Web interactive, downloadable
**Access:** [ACLED Conflict Index](https://acleddata.com/series/acled-conflict-index)
**License:** Non-commercial academic use
**Last Verified:** April 2026

---

## Description

A composite index derived from ACLED event data that ranks countries by current conflict
intensity. Distinct from the raw ACLED event dataset — this is a pre-aggregated country-level
summary score, useful for quick cross-country comparison without querying the full event database.

## Merge Keys & Compatibility

- Merge with raw **ACLED** data and **GPI** via `iso3c`
