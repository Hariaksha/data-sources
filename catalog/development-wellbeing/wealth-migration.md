# Henley Private Wealth Migration Report

**Theme:** Economics & Development
**Source:** Henley & Partners
**Geographic Coverage:** Global (selected wealth markets)
**Time Coverage:** Annual
**Unit of Observation:** Country
**Format:** PDF report, web
**Access:** [Henley Private Wealth Migration Report 2025](https://www.henleyglobal.com/publications/henley-private-wealth-migration-report-2025/the-worlds-fastest-growing-wealth-markets)
**License:** Free to access
**Last Verified:** April 2026

---

## Description

Tracks the number of millionaires, centi-millionaires, and billionaires per country along with
10-year millionaire growth rates. Useful for understanding wealth concentration dynamics and
identifying fast-growing economies from a private wealth perspective.

## Notes

- Complementary to inequality data (WIR, MPI) — shows the top of the distribution
  while MPI/WIR cover the bottom and middle

## Merge Keys & Compatibility

- Merge with **World Inequality Report**, **HDI** via `iso3c`
