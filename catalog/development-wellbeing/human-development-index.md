# Human Development Index (HDI)

**Theme:** Development
**Source:** UNDP Human Development Reports Office
**Geographic Coverage:** Global (~193 countries)
**Time Coverage:** 1990 – present (annual)
**Unit of Observation:** Country-year
**Format:** Downloadable CSV, API
**Access:** [UNDP HDR Data](https://hdr.undp.org)
**License:** Free to access
**Last Verified:** April 2026

---

## Description

Composite index combining life expectancy, education (mean and expected years of schooling),
and GNI per capita. The standard benchmark for human development across countries. Widely
used as a control or outcome variable in development economics.

## Key Variables

| Variable | Description |
|----------|-------------|
| HDI score | Composite 0–1 score |
| Life expectancy index | Health dimension |
| Education index | Combined schooling dimension |
| GNI index | Income dimension |

## Merge Keys & Compatibility

- Merge with almost any country-level dataset via `iso3c`
- Annual coverage from 1990 enables long panel analysis
